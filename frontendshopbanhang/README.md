# Frontend (React Vite) for Shopbanhang (mock)

## Setup

1. Install dependencies:

```bash
cd frontend
npm install
```

2. Run dev server:

```bash
npm run dev
```

The dev server runs on `http://localhost:3001`. API requests to `/api` are proxied to backend `http://localhost:8080`.

## Notes
- Uses mock data in `src/services/mockApi.js`. Replace calls with real API endpoints later.
- Simple layout and pages: Home, ProductDetail, Cart.
